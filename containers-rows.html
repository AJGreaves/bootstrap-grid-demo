Containers & Rows

  • Containers and rows only have width properties.
  • They have no height of their own.
  • Their height is determined by the height of their columns content.

Containers

  • There are two container classes in the bootstrap library
    1. container
    2. container-fluid
  • The difference between these two classes is the width of the screen that the container takes up
  • The width of a container often does not take up the whole width of the screen
  • It usually leaves a lot of margin to the sides depending on the screen size
  • The width of a container-fluid is always 100% of the width of the screen
  • Containers can be nested inside other elements, in this case their width will be relative to the width of the element they are inside.
